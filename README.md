# Мережне програмування

## Опис проекту
Проект `LabsMP` містить лабораторні роботи з курсу "Мережне програмування".

## Структура проекту
Репозиторій організовано у вигляді окремих папок для кожної лабораторної роботи та кореневих файлів для налаштувань:

### Lab1
Папка з лабораторною роботою №1, яка містить:
- Підпапку `screens` зі скриншотами виконаних завдань та графіками.
- Python-скрипти.
- Файл `requirements.txt` з переліком залежностей.
- README.md з описом ходу виконання роботи.

### Lab2
Папка з лабораторною роботою №2.

### Lab3
Папка з лабораторною роботою №3.

### Кореневі файли
- `.gitignore` — файл для ігнорування непотрібних файлів та папок у репозиторії.
